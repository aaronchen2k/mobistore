# eCommerce Fashion Store

Ionic 2.0, AngularJS 2.0, Node.js, MongoDB

### Quick start
```bash
Install mongodb-org-3.0
Import MongoDB data from xdoc/mongodb-dump.zip

cd mobistore
sudo npm install
npm run dev

cd client
sudo npm install

ionic serve
```
